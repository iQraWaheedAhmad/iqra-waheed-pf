# Iqra Waheed — Portfolio

Built with Next.js (App Router), TypeScript, Tailwind CSS v4, and GSAP.

## Getting started

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Before you deploy

Edit `src/lib/data.ts`:
- Add your real LinkedIn, GitHub, and portfolio URLs (`profile.linkedin`, `profile.github`, `profile.portfolio`).
- Add live/GitHub links to each project once available (`liveUrl`, `githubUrl` on each project).

## Deploy

Push to GitHub and import into Vercel, or run:

```bash
npm run build
npm run start
```
